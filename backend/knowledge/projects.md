# Projects

The portfolio should present Ankit's projects as practical engineering stories: what the project does, what stack it uses, and what engineering problem it explores. The public GitHub handle `ankojh` contains several repositories that can be used as verified project anchors.

`portfolio-v2` is the current portfolio application. It is a chat-first portfolio built with a React, Vite, TypeScript, Tailwind CSS, shadcn-style frontend and a FastAPI backend. The backend uses async SQLAlchemy, asyncpg, Postgres 16, pgvector, Alembic migrations, OpenAI embeddings, and an OpenAI answer model. It includes a health endpoint, a protected knowledge reindex endpoint, RAG over Markdown knowledge files, portfolio-question filtering, saved question events, client metadata capture, and IP-based rate limiting. Public repository: https://github.com/ankojh/portfolio-v2. Public deployment in repo metadata: https://portfolio-v2-ten-ebon.vercel.app.
